# BookStore
ExpressJS, MongoDB, ReactJS 

# Start api-bookstores
````
cd api-bookstores/
npm install
npm run start
````
# Start react-bookstores
````
cd react-bookstores/
npm install
npm run start
````

Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
